Instructions

  • Select IP version (IPv4 or IPv6)
  • Choose IP class for IPv4 (Class A-E or Any)
  • Enter the number of IPs to generate (1-100)
  • Option to ensure uniqueness or allow duplicates
  • Click "Generate" to create IP addresses

IPv4 Address Classes

  • Class A: 0.0.0.0 - 127.255.255.255 (Large networks)
  • Class B: 128.0.0.0 - 191.255.255.255 (Medium networks)
  • Class C: 192.0.0.0 - 223.255.255.255 (Small networks)
  • Class D: 224.0.0.0 - 239.255.255.255 (Multicast)
  • Class E: 240.0.0.0 - 255.255.255.255 (Reserved)

IPv6 Format

IPv6 addresses use hexadecimal notation in 8 groups of 4 hex digits, separated by colons (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334).
