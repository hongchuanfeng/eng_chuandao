IP Geolocation Lookup

Enter an IP address to search, or click "Get My IP"

About IP Addresses:

  • 1. An IP address (Internet Protocol Address) is a numeric label that identifies devices on the internet.
  • 2. IP addresses define how devices address and communicate with each other in packets.
  • 3. There are two main types: IPv4 and IPv6.
  • 4. This tool looks up the geolocation of an IP. Results are for reference only and not for commercial use.

How to Use IP Geolocation Lookup

Find the geographical location and information for any IP address instantly.

Instructions

  • Enter an IP address (e.g., 192.168.1.1) in the search box
  • Click "Search" to query the IP location
  • Or click "Get My IP" to automatically look up your current IP address
  • View detailed information including country, city, ISP, timezone, and more

What Information is Displayed?

  • Country & City: Geographic location of the IP
  • ISP: Internet Service Provider
  • Timezone: Local time zone information
  • Latitude & Longitude: Precise coordinates
  • Organization: Associated organization

Examples

Example 1: Query "8.8.8.8" to see Google's DNS server location.

Example 2: Query "1.1.1.1" to see Cloudflare's DNS server location.

Privacy Note

IP geolocation is approximate and based on database lookups. Results are for reference only and may not represent the exact physical location of a device. For privacy reasons, we do not store or track any IP addresses used in queries.
