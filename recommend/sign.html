How to Use the Electronic Signature Generator

Create digital signatures for documents, contracts, and official paperwork with this interactive tool.

Instructions

  • PC Users: Hold right mouse button and drag to write your signature
  • Mobile Users: Touch and drag on the screen to create your signature
  • Use the toolbar to customize colors and export options
  • Clear the canvas to start over if needed

Toolbar Functions

  • Pen Color: Choose the color for your signature
  • Background Color: Set the background color for exported images
  • Base64: Copy the signature as Base64 encoded data
  • Download PNG: Save signature as PNG image file
  • Download SVG: Save signature as scalable vector graphics
  • Undo: Remove the last stroke you drew
  • Clear: Erase the entire signature

Export Formats

  • PNG: Raster image with transparent or colored background
  • SVG: Vector format that scales without quality loss
  • Base64: Text format for embedding in web pages or documents

Technical Notes
