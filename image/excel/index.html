How to Use

What this tool does

Convert an image into an Excel workbook with colored cells (pixel art). Tune image adjustments and Excel cell options for best results, then export to .xlsx.

Steps

  1. Upload: Click/drag an image into the upload area.
  2. Adjust Image: Set Max pixels (resolution), tweak Brightness, Contrast, Saturation, and Hue.
  3. Excel Settings: Choose Cell size, sheet background, cell border style/color, and Zoom.
  4. Preview: Review the pixel grid preview on the right.
  5. Generate: Click Generate Excel file to download an .xlsx file.

Tips

  • Higher Max pixels ‚Üí more detail but larger Excel and slower generation.
  • Use small Cell size for finer pixel art; larger for mosaic style.
  • Borders help readability but increase file size.

Limits & Privacy

  • Processing runs locally in your browser; images are not uploaded.
  • Very large images can be slow; reduce Max pixels if needed.
  • Excel rendering time depends on the number of colored cells.

Browser Requirements

  • Modern browsers (Chrome, Edge, Firefox, Safari) with JavaScript enabled.
