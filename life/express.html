Tracking Information Displayed

  • Tracking Number: Your package's unique identifier
  • Courier Company: The delivery service provider
  • Current Status: Latest delivery status
  • Tracking History: Chronological list of package movements
  • Timestamps: When each status update occurred

Common Status Types

  • In Transit: Package is being transported
  • Collected: Package has been picked up
  • Exception: Delivery issue or delay
  • Delivered: Package successfully delivered
  • Out for Delivery: Package is out for final delivery
  • Returned: Package returned to sender
  • Transferred: Package transferred between facilities

Common Use Cases

  • Track online shopping deliveries
  • Monitor business shipments
  • Check package delivery status
  • Verify delivery completion
  • Resolve delivery issues
  • Plan package pickup

Tips for Better Results

  • Enter the complete tracking number without spaces
  • Use auto-detect for convenience, manual selection for accuracy
  • Check tracking numbers are correct before searching
  • Some packages may take time to appear in tracking systems
  • Contact the courier company for urgent delivery issues

Error Messages

  • "Please enter a tracking number": No input provided
  • "Unable to identify the courier": Auto-detect failed, select manually
  • "No tracking info found": Package not found or not yet in system
  • "Interface does not support this courier": Try a different courier selection
  • "Query failed, please try again later": Network or service issue

Privacy & Security

  • Tracking queries are processed through secure APIs
  • No personal information is stored
  • Tracking numbers are only used for status queries
  • Results are displayed locally in your browser
