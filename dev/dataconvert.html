About data storage units:

  1. bit: the smallest data unit representing 0 or 1
  2. Byte: 1 Byte = 8 bits
  3. KB: 1 KB = 210 Bytes = 1,024 Bytes
  4. MB: 1 MB = 220 Bytes = 1,048,576 Bytes
  5. GB: 1 GB = 230 Bytes = 1,073,741,824 Bytes
  6. TB: 1 TB = 240 Bytes = 1,099,511,627,776 Bytes
  7. PB: 1 PB = 250 Bytes
  8. EB: 1 EB = 260 Bytes
  9. ZB: 1 ZB = 270 Bytes
