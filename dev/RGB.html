About this tool:

  1. RGB (red, green, blue) is an additive color model.
  2. Hex color codes are widely used on the web and in graphics.
  3. Convert between RGB values and hex color codes in real time.

How to Use

What this tool does

Convert between RGB values and HEX color codes instantly, and preview the color in real time.

Steps

  1. Enter R, G, B values (0–255).
  2. Color preview and HEX update automatically.
  3. Copy the HEX code for use in your projects.

Examples

  • RGB(255, 0, 0) → #FF0000
  • RGB(0, 130, 119) → #008277

Notes

  • Inputs are clamped to 0–255 for validity.
  • HEX output is uppercase for readability.
