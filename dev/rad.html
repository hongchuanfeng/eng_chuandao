All Radix Converter

How to Use

What this tool does

Convert numbers among binary, octal, decimal, and hexadecimal with validation and clear results.

Steps

  1. Choose the base of your input number.
  2. Enter the number and click Convert.
  3. View results in all supported bases.

Examples

  • Binary 1010 → Decimal 10, Hex 0xA.
  • Decimal 255 → Hex 0xFF, Binary 11111111.

Notes

  • Input is validated per base; only valid characters are accepted.
  • Hex output is uppercase and binary padded to 8 bits for readability.
