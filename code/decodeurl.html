DecodeURI Decoder

About this tool:

  • DecodeURI: Decodes a URI encoded with encodeURI, converting percent-encoded characters back to their originals.
  • DecodeURIComponent: Decodes a URI component encoded with encodeURIComponent, including reserved characters.

How to Use

What this tool does

Decodes URL-encoded content using decodeURI or decodeURIComponent.

When to use

  • decodeURI: full URL string.
  • decodeURIComponent: a single parameter value or path segment.
