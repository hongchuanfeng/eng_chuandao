How to Use

What this tool does

Converts an image file to a Base64 data URL and shows the HTML <img> snippet for quick embedding.

Steps

  1. Click the upload area or drag an image onto it.
  2. Copy the Base64 data or the generated HTML snippet.
  3. Use Clear to reset and convert another image.

Examples

  • Result starts like data:image/png;base64,iVBORw0KGgoAAAANSUhEUg...
  • HTML: <img src="data:image/jpeg;base64,/9j/4AAQ..." alt="Base64 Image">

Notes

  • Very large images create long strings; consider compressing before embedding.
  • Base64 increases size by ~33%; use only when inline embedding is needed.
  • For privacy, processing happens locally in your browser.
